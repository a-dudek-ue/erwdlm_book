# Deep Learning Models for Economic Research

This repository contains code, datasets, and supplementary materials accompanying the forthcoming "Deep Learning Models for Economic Research" by Routledge (https://www.routledge.com/Deep-Learning-Models-for-Economic-Research/Dudek/p/book/9781041062707)
It is intended for researchers, students, and data scientists applying modern deep learning and explainable AI techniques in economic and financial contexts.
